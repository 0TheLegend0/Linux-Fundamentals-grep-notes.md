#Windows Fundamentals 
- New Technology File System or NTFS
      New and improved way of filing data
        The improvements from fat16/fat32 abd HPFS
          - Larger than 4Gb files
          - Ability to set desired permissions on folders and files
          - Ability to compress folders and files
          - Ability to use EFS(Encrytption File System)

- User Account Control or UAC
      If the Admin on a system gives you only privileges when needed, by prompting to stop unauthorized access
